INSTRUCCIONES

El juego consiste en acertar las veintisiete palabras antes ques se termine el tiempo, cada una de las cuales se corresponde con una letra del rosco.
Los aciertos se reflejarán en las letras mediante el color verde, mientras que los fallos se mostrarán en color rojo.
Las cuestiones no respondidas o "pasapalabra" se mostrarán en color azul.

Escribe la respuesta correctamente, con todas las letras. Las tildes no son necesarias, el programa las ignora.
El programa detecta la tecla intro. Respuesta vacia es "Pasapalabra". Respuesta lletra "Comprobar"
